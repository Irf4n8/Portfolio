About

I'm a Computer Science Engineering graduate from Coimbatore, Tamil Nadu, with hands-on experience building software through two internships and two end-to-end projects. My core stack starts with C++ for problem-solving and fundamentals, followed by Python and FastAPI on the backend, with a growing command of REST API design, database management using MySQL and Supabase, and enough frontend basics in HTML, CSS, JavaScript, and Figma to ship a feature start to finish.

At Fnext Solutions, I worked on responsive web applications and REST API integration alongside senior developers in an agile team. At Vulture Lines Tech Management, I built REST APIs and dashboards for real-time IoT sensor data, working closely with hardware engineers to bring software and embedded systems together. That experience carried straight into AgriTwin, a digital twin platform giving farmers real-time monitoring and predictive insights, and Travel-X, a travel experience-sharing app built with proper authentication and validation middleware.

I bring a solid grounding in system design, data modeling, and debugging real production issues plus the habit of learning whatever the next problem needs. I'm actively looking for a full-stack or backend role where I can contribute from day one and keep growing fast.
