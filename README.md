# Mohammed Irfan - Interactive Neo-Brutalist Portfolio

A high-fidelity developer portfolio showcasing backend engineering expertise, custom liquid animations, and interactive mapping.

## Overview
This is an interactive, responsive developer portfolio designed with a striking neo-brutalist aesthetic. It showcases project work, professional experience, and technical skills using custom layouts, interactive maps, and fluid animations.

### Features
- **Smart Sticky Navigation**: Top navigation bar with smooth scrolling and precise offset alignment.
- **Ink Spill Dark Mode**: A canvas-based, organic theme toggle resembling liquid ink expanding and receding across the screen.
- **Skills Grid with Interactive Icons**: Custom inline SVG category icons that trigger looping, micro-animations on hover.
- **Interactive Mapping**: Leaflet-powered map displaying location markers for past education and internship roles.
- **Tiled Canvas Background**: Subtle repeating paper fabric grain overlay texture.

## Tech Stack
- **Libraries**: Leaflet.js (Map), FontAwesome & Devicon (Brand logos)
- **Styling**: Vanilla CSS3
- **Logic**: Vanilla Javascript (ES6+, Canvas 2D Context)
